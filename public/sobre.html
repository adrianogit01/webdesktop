Sobre o Web Desktop

O Web Desktop é um projeto experimental desenvolvido com HTML, CSS e JavaScript que simula um ambiente de desktop diretamente no navegador.

O objetivo é fornecer uma interface familiar e produtiva, integrando aplicativos como calculadora, player de mídia, navegador, gerenciador de arquivos e mais — tudo rodando sem a necessidade de instalações.

Este sistema é responsivo, compatível com dispositivos móveis e pode ser usado como base para PWAs, sistemas educacionais ou demonstrações de UI/UX.

Se vc está usando no celular e de alguma forma espelhando em uma tela maior, coloque o navegador do celular em modo desktop e arraste uma das janelas para a direita (pode ser essa mesmo) o que faz um zoom out, assim o aproveitamento do desktop é melhor.
